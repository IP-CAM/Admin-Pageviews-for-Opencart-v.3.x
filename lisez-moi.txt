Pour installer l'extension Pageviews, il suffit de téléverser le dossier admin
à la racine du dossier du site Web OpenCart, sur le serveur de destination.

Une fois les fichiers téléversés sur le serveur de destination,
il faudra se rendre dans l'interface d'administration pour :
    1. Installer l'extension                                    Extensions > Extensions > Report
    2. Téléverser le dossier compressé pageviews.ocmod.zip      Extensions > Installer
    3. Rafraîchir le cache des modifications                    Extensions > Modifications

Pour désinstaller l'extension Pageviews, il faudra se rendre dans l'interface d'administration pour :
    1. Désinstaller l'extension                                 Extensions > Extensions > Report
    2. Désinstaller le fichier de Modifications                 Extensions > Installer
    3. Rafraîchir le cache des modifications                    Extensions > Modifications
